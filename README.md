# 🎬 Tamil OTT Bias-Corrected Content Performance Prediction

## 📌 Project Overview
This project focuses on predicting the performance of Tamil OTT content using a bias-corrected machine learning approach. Traditional systems are biased towards mobile user data, ignoring Smart TV and rural audience contributions.

## 🚨 Problem Statement
Mobile usage dominates OTT analytics, while Smart TV and rural users are underrepresented. This leads to unfair and inaccurate content performance prediction.

## 💡 Solution
We developed a data-centric ML model that:
- Applies bias correction to balance mobile and TV data
- Includes rural audience impact
- Predicts content performance using machine learning

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## 🔄 Workflow
Data Generation → Preprocessing → Bias Correction → Model Training → Prediction → Evaluation

## 📊 Features Used
- Mobile Views
- TV Views
- Completion Rate
- Rural Ratio
- YouTube Views

## 🤖 Model
Random Forest Regressor is used to predict the content performance score.

## 📈 Results
- Improved fairness in prediction
- Better performance using bias-corrected data
- Visualization of Actual vs Predicted results
